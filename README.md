# Amazon E-commerce Data Analysis

This project explores an Amazon e-commerce dataset from Kaggle to practice data cleaning, SQL analysis, and data engineering-style workflows.

## Goals

- Clean and validate raw e-commerce product data
- Explore product categories, ratings, prices, and discounts
- Create SQL queries for business-style analysis
- Build a small portfolio project using real-world data

## Dataset

Source: Kaggle - Amazon E-commerce Dataset

The raw dataset is not included in this repository. Please download it directly from Kaggle.

## Planned Steps

1. Download and inspect the dataset
2. Clean missing, duplicate, and inconsistent values
3. Load the data into SQL Server
4. Write analytical SQL queries
5. Document insights and results
